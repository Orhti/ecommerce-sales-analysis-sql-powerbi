# E-commerce Sales Analysis (SQL + Power BI)

## Project Overview

This project analyzes 100k+ e-commerce transactions to identify revenue trends, product performance, and customer purchasing behavior using SQL and Power BI.

## Tools Used

- MySQL
- Power BI
- SQL
- Excel

## Key Insights

- Credit cards account for 73% of transactions
- São Paulo generates the highest revenue
- Beauty & Health is the top product category
- Revenue peaks during May and August
